## What I understood
- Analytic functions allow calculations across multiple rows while keeping each row in the result.
- Unlike aggregate functions, analytic functions do not reduce the number of rows.
- The OVER() clause defines the window of rows used for the calculation.
- PARTITION BY splits the data into groups where the analytic function is applied independently.
- ORDER BY inside the window defines the logical order used for calculations such as ranking or running totals.
- Windowing clauses like ROWS BETWEEN ... define which rows relative to the current row are included in the calculation.
- A sliding window allows calculations such as moving averages or running totals.

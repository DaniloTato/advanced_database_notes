## What I understood
- Set operators combine the results of multiple SELECT queries into a single result set.
- For set operators to work, each query must return: the same number of columns.
- UNION merges results and removes duplicate rows.
- UNION ALL merges results but keeps duplicates.
- MINUS returns rows that exist in the first query but not the second.
- INTERSECT returns rows that exist in both queries.
